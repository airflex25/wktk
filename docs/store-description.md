• 자전거·등산·드라이브 중 잠깐 짧게 말하고 싶은 분
• 게임 중 마이크 켜놓고 친구와 수다 떨고 싶은 분
• 일반 전화/메신저 음성 대신 가벼운 라디오식 통화를 원하시는 분

━━━━━━━━━━━━━━━━━━━━━━━━━━━

  같은 6자리 숫자를 입력한 사람끼리 즉시 같은 채널.
  복잡한 ID, 친구 추가, 전화번호 공유 불필요.

▪ PUSH TO TALK / 항상 켜기
  버튼 누른 동안만 송신하는 워키토키 모드(PTT)
  마이크 항상 ON으로 두는 VOX 모드 (단체 대화에 편함)

▪ 8명까지 동시 대화
  같은 키 룸에 최대 8명까지 동시 참여.

▪ 가입 없음 / 광고 없음 / 추적 없음
  계정도, 이메일도 필요 없어요.
  광고 ID도 안 씁니다. 분석 도구도 안 씁니다.

▪ 음성은 P2P
  통화는 WebRTC로 단말기끼리 직접 연결.
  목소리는 서버를 거치지 않으며 어디에도 저장되지 않습니다.

## 데이터 안전 양식 답안

| 질문 | 답 |
|------|-----|
| 데이터를 수집/공유합니까? | **아니요** |
| 모든 데이터가 전송 중에 암호화됩니까? | **예** (HTTPS/WSS, DTLS-SRTP) |
| 사용자가 데이터 삭제를 요청할 수 있습니까? | **해당없음** (수집하는 데이터 없음) |
| 데이터 처리 관행이 Play 가족 정책을 준수합니까? | **예** |

권한 사용 사유:
- **마이크**: "통화 음성 전송에 사용. 녹음하지 않으며 서버에 전송되지 않음"
- **인터넷/네트워크 상태**: "통화 연결을 위한 시그널링 통신"
